How It Works

Audio files are loaded using the HTML5 <audio> element.

JavaScript controls playback, track switching, and progress updates.

CSS handles layout, responsiveness, and visual styling.

User actions (play, pause, seek) are handled through JavaScript event listeners.

How to Run

Download or clone the repository.

Open index.html in any modern web browser.

Select a track and start listening.

No backend or database setup is required.

Limitations

Front-end only (no user authentication or streaming backend)

Music files are stored locally

No playlist saving or account system
